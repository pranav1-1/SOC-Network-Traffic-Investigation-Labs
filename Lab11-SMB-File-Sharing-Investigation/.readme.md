# Lab 11 – SMB File Sharing Investigation

## Objective

The objective of this lab is to understand how the Server Message Block (SMB) protocol works by creating and analyzing file-sharing traffic between a Kali Linux client and an Ubuntu Samba server using Wireshark.

This lab demonstrates how Security Operations Center (SOC) analysts investigate SMB traffic, identify file access operations, and detect potential lateral movement or unauthorized file activity.

---

## Scenario

The SOC team received an alert indicating file access activity on an internal file server. Your task is to investigate the SMB traffic and determine whether the observed activity is legitimate.

As the assigned SOC Analyst, you will analyze the network traffic to identify:

- Source and destination hosts
- SMB session establishment
- Shared folder access
- File creation, read, write, rename, and deletion operations
