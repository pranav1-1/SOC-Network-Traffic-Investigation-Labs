# Lab 1 – Environment Setup

## Objective
Set up a SOC Analyst home lab using Kali Linux, Ubuntu Server, VirtualBox, and Wireshark.

## Lab Environment

- Kali Linux
- Ubuntu Server
- VirtualBox
- Wireshark

## Tasks Performed

- Installed Kali Linux
- Installed Ubuntu Server
- Configured VirtualBox networking
- Verified network connectivity
- Installed Wireshark
- Verified packet capture

## Outcome

The lab environment was successfully configured and is ready for network traffic analysis using Wireshark.
